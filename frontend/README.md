# To Do Pixel

**To Do Pixel** es una aplicación web para gestionar tus tareas diarias de manera simple y visual, con un estilo  único. Permite crear, editar, eliminar y filtrar tareas, con una interfaz atractiva y fácil de usar.

## Características

- Crear nuevas tareas con título y descripción.
- Marcar tareas como completadas o pendientes.
- Editar y eliminar tareas existentes.
- Filtrar tareas por estado (completadas / pendientes).
- Autenticación con JWT para usuarios registrados.

## Tecnologías

- **Frontend:** React, CSS.
- **Backend:** Node.js, Express
- **Base de datos:**  MongoDB 
- **Autenticación:** JWT
- **Testing:** Jest
